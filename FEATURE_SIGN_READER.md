# 🔍 Sign Reader (Mobile-First)

## Goal
Help users quickly understand confusing parking signs from a photo.

## Recommended for mobile web
Use your own Claude API key in-session (optional field in the app).

- No account required for app usage in general
- Account only needed if user wants AI sign analysis
- API key is never stored by SignSpot

## Privacy
- Photos are processed for analysis only
- No photo storage
- No API key storage
- No personal data persistence

## Modes
1. **Claude mode (recommended for mobile):**
   - Better multilingual accuracy
   - Works well on phone browsers
   - User pays from their own API account

2. **Local Ollama mode (optional desktop fallback):**
   - No API costs
   - Requires local model setup
   - Not ideal for mobile users

## Simple user flow
1. Open Sign Reader
2. (Optional) paste personal Claude API key
3. Upload sign photo
4. Get plain-language explanation

## Status
MVP ready. Keep lightweight and privacy-first.
